## FolderKeywordRename
根据文件夹包含的关键字来重命名至指定的名称</br>只需文件夹包含需要的关键字，即可重命名至指定的名称
## 注意 (Note)
本存储库中的 cpp 文件是由 deepseek 生成的</br>(The cpp files in this repository were generated by deepseek.)
## 如何使用 (how to use)
<code>FolderKeywordRename.exe -k keyword -n replace</code></br>将 keyword 替换成需要的关键词，将 replace 替换成需要的文件夹名称</br>(Replace "keyword" with the desired keyword, and replace "replace" with the desired folder name.)
## 自己下载源代码然后编译
1.在存储库首页单击"Code"，然后"Download ZIP"，将下载后的压缩文件解压缩至合适位置</br>2.切换至对应的文件夹：在 <code>命令提示符中使用 cd 命令切换对应文件夹 </code> 和 <code> 资源管理器跳转到对应文件夹并在地址栏中使用 cmd</code> 均可</br>3.使用 <code>g++ -std=c++17 -m64 -o FolderKeywordRename.exe FolderKeywordRename.cpp -lshlwapi -static</code> 运行编译.</br>
